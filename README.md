## How to install

1. Clone project
2. Install Composer

        composer install
3. Copy `.env.example` and set values
4. Create Laravel key

        php artisan key:generate
5. Run migration

        php artisan migrate

6. Run DB Seeder

        php artisan db:seed
7. Install npm packages
        
        npm install
8. Serve Project

        npm run dev
